# Online Survey System

A Flask-based Online Survey System developed for a B.Tech project.

## Features

- User Registration & Login
- Create Dynamic Surveys
- Rating Questions
- Yes/No Questions
- Text Questions
- Multiple Choice Questions (MCQ)
- Survey Analytics
- PDF Report Download
- Excel Report Download

## Technologies Used

- Python
- Flask
- MySQL
- Bootstrap 5
- Pandas
- Matplotlib
- ReportLab
- OpenPyXL
